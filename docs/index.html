C++20 Minecraft Bedrock Protocol Library

Build fast Minecraft Bedrock bots in modern C++.

A C++20 library for Minecraft Bedrock Edition with RakNet networking, Xbox authentication, packet serialization, relay tools, chunk foundations and version registry support.

View on GitHub Read Docs
example.cpp
#include <bedrock/bedrock.hpp>

int main() {
    auto client = bedrock::createClient({
        .host = "localhost",
        .port = 19132,
        .username = "Bot",
        .version = "1.20.40",
        .offline = true
    });

    client.on("start_game", [](const bedrock::Packet&) {
        std::cout << "Joined world\n";
    });

    return client.run();
}

Installation

Required tools: C++20 compiler, CMake, Ninja or Make, OpenSSL, zlib and curl.

git clone https://github.com/M9CHKO/bedrock-protocol-cpp.git
cd bedrock-protocol-cpp
./scripts/build.sh

Client API

Create a Bedrock client with host, port, username, version and auth mode.

auto client = bedrock::createClient({
    .host = "cpe.ign.gg",
    .port = 19132,
    .username = "StewedV",
    .version = "1.20.40",
    .offline = false
});

client.run();

Events

Subscribe to packet and connection events.

client.on("connect", []() {
    std::cout << "Connected\n";
});

client.on("start_game", [](const bedrock::Packet& packet) {
    std::cout << "Start game received\n";
});

client.on("disconnect", []() {
    std::cout << "Disconnected\n";
});

Packet Writing

Send packets by name or id using the protocol registry.

client.write("request_chunk_radius", {
    {"chunk_radius", 8},
    {"max_radius", 8}
});

client.write("text", {
    {"type", "chat"},
    {"message", "Hello from C++ bot!"}
});

Authentication

Offline mode is useful for local test servers. Online mode uses Xbox authentication.

// Local / offline server
.offline = true

// Public online server
.offline = false

Relay API

Relay mode can inspect, forward, cancel or replace packets between client and server.

relay.onClientbound("text", [](auto& packet) {
    std::cout << "Server message\n";
});

relay.onServerbound("move_player", [](auto& packet) {
    std::cout << "Player moved\n";
});

Simple Server

The repository also contains simple server examples for testing and protocol experiments.

auto server = bedrock::createServer({
    .host = "0.0.0.0",
    .port = 19132,
    .motd = "C++ Bedrock Server"
});

server.listen();

Project Structure

  • include/bedrock — public headers.
  • src — core implementation.
  • examples — ready-to-run examples.
  • docs — documentation files.
  • data — protocol and registry data.
  • scripts — build scripts.

FAQ

Can it connect to public servers?

Yes, online mode is intended for real Bedrock servers with Xbox authentication.

Can it be used in Termux?

Yes, if the required C++ build tools and libraries are installed.

Does it support many Minecraft versions?

Yes, the library includes a version registry and protocol mappings for many Bedrock versions.

Is it only a bot library?

No. It can also be used for packet debugging, relay tools, protocol experiments and server-side tests.
