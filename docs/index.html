Project Overview

Minecraft Bedrock Protocol Client Library for C++20 with Xbox Authentication, RakNet Networking and Packet Serialization.

🌐

RakNet Networking

Ping, connect, handshake and MCPE packet transport foundation.

🔐

Xbox Authentication

Online login with profile cache and interactive device-code auth.

📦

Packet Serialization

Schema-shaped packet reading and writing across bundled versions.

🔁

Relay Runtime

Inspect, cancel, replace and forward packets between client and server.

🗺️

World / Chunks

Level chunk decoding foundation and tracked client world state.

🧩

Version Registry

Bundled protocol data for many Minecraft Bedrock versions.
